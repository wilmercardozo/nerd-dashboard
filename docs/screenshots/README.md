# Capturas

Pon aquí las imágenes que muestra el README principal. No puedo generarlas yo
(requieren la placa física y el dashboard web corriendo).

## Qué capturar

| Archivo | Qué |
|---------|-----|
| `device.jpg` | Foto del CYD encendido mostrando una vista (de cerca, buena luz). |
| `web-dashboard.png` | Captura del dashboard web (`http://<ip>:8765`) completo. |
| `view-servicios.jpg` | Tab Servicios en el display. |
| `view-tokens.jpg` | Tab Tokens (gauges 5h/semana). |
| `view-pc.jpg` | Tab PC (anillo CPU + métricas). |
| `view-tareas.jpg` | Tab Tareas (pomodoro + lista). |

## Tips

- Fotos del display: ~1200px de ancho, recorta el marco, evita reflejos.
- Capturas web: navegador a ~1280px, tema oscuro.
- Un GIF corto (`demo.gif`) del pomodoro o de un servicio cayéndose vende mucho
  el repo en la cabecera del README.

Cuando las subas, avísame y conecto las referencias en el README.
