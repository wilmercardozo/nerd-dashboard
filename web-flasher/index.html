⚡ Nerd Dashboard

Flashea tu placa ESP32-2432S028R (CYD) desde el navegador. Sin PlatformIO.

Tu navegador no soporta Web Serial. Usa Chrome o Edge de escritorio. Debe servirse por HTTPS (o localhost).
  1. Conecta la placa por USB.
  2. Pulsa Conectar y flashear y elige el puerto serie de la placa.
  3. Espera a que termine; la placa reinicia sola.
  4. Primer arranque → red WiFi NerdDashboard-Setup → abre http://192.168.4.1 y configura WiFi + IP del agente.

Solo Chrome/Edge de escritorio (Web Serial). Tras flashear, instala el agente PC.
